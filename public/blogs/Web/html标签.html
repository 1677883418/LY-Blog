    # html

    • 基础标签
      • <!DOCTYPE> (opens new window)
        • 定义文档类型
      • <html> (opens new window)
        • 定义一个HTML文档
      • <head> (opens new window)
        • 定义页面头部内容,元数据meta等必须包含title标签
        • <meta> (opens new window)
          • 描述文档元数据
            • charset:设置页面编码
            • content:定义与 http-equiv 或 name 属性相关的元信息。
            • http-equiv:把 content 属性关联到 HTTP 头部.
            • name:把 content 属性关联到一个名称。
            • scheme:HTML5不支持 定义用于翻译 content 属性值的格式。
        • <title> (opens new window)
          • 定义页面标题 浏览器标签页
      • <body> (opens new window)
        • 定义页面主体部分
        • <h1~h6> (opens new window)
          • 定义HTML页面标题
        • <p> (opens new window)
          • 段落标签
          • align
            • HTML5不支持。 HTML 4.01已废弃, 不赞成使用。 规定段落中文本的对齐方式。
        • <br> (opens new window)
          • 换行符,输入空行，而不是分割段落
        • <hr> (opens new window)
          • 定义水平线
          • align
            • HTML5 不支持。HTML 4.01 已废弃。 规定元素的对齐方式
          • noshade
            • HTML5 不支持。HTML 4.01 已废弃。 规定元素的颜色呈现为纯色。
          • size
            • 规定元素高度
          • width
            • 规定元素宽度
        • <!----> (opens new window)
          • 注释
    • 格式标签
      • <strong> (opens new window)
        • 定义粗体
      • <s> (opens new window)
        • 删除线
      • <u> (opens new window)
        • 下划线
    • 表单
      • <form> (opens new window)
        • 定义一个 HTML 表单，用于用户输入。
      • <input> (opens new window)
        • 定义一个输入控件
    • 列表
      • <ul> (opens new window)
        • 定义一个无序列表
      • <ol> (opens new window)
        • 定义一个有序列表
      • <li> (opens new window)
        • 定义一个列表项
      • <dl> (opens new window)
        • 定义一个定义列表
      • <dt> (opens new window)
        • 定义一个定义定义列表中的项目
      • <dd> (opens new window)
        • 定义定义列表中项目的描述
      • <menu> (opens new window)
        • 定义菜单列表
    • 表格
      • <table> (opens new window)
        • 定义一个表格
      • <caption> (opens new window)
        • 定义表格标题
      • <th> (opens new window)
        • 定义表格中的表头单元格
      • <tr> (opens new window)
        • 定义表格中的行
      • <td> (opens new window)
        • 定义表格中的单元
      • <thead> (opens new window)
        • 定义表格中的表头内容
      • <tbody> (opens new window)
        • 定义表格中的主体内容
      • <tfoot> (opens new window)
        • 定义表格中的表注内容(脚注)
      • <col> (opens new window)
        • 定义表格中一个或多个列的属性值
      • <colgroup> (opens new window)
        • 定义表格中供格式化的列组
